# Blog

這是一個使用 **ASP.NET Core MVC** 開發的個人部落格系統。

* **後端框架：** ASP.NET Core MVC .NET 8
* **程式語言：** C#
* **前端技術：** HTML5, CSS3, Bootstrap 5, jQuery
* **資料庫：** SQL Server / Entity Framework Core
* **版本控制：** Git & GitHub

其中包含
* **文章管理：** 支援文章的建立、讀取、更新與刪除 (CRUD 操作)。
* **依賴注入 (DI)：** 善用 ASP.NET Core 內建的 DI 容器管理服務生命週期。
* **實作分頁功能、身分驗證及驗證碼輸入、串接ckEditer**
